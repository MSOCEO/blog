## 欢迎

这是叙溯 Xusu v2.0 的第一篇文章。

### 技术栈

- **Next.js 15** — React 框架，静态导出
- **TypeScript** — 类型安全
- **Tailwind CSS 4** — 原子化 CSS
- **gray-matter + remark/rehype** — Markdown 处理

### 设计主题

极光 Aurora 主题采用混搭风格：

- **mrzym.top** 的博客页面结构（时间轴 / 分类 / 说说）
- **koel.dev** 的深色质感（`#080810` 深海黑 + `#E91E8C` 品红）

### 代码示例

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```

> 这是一个引用块。Xusu 支持 GFM 表格、删除线等扩展语法。
